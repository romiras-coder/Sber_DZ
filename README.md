### Тестовое задание

Расчет депозита

#### Пример входного вектора

    {
        "date": "31.01.2021",
        "periods": 10,
        "amount": 10000,
        "rate": 8
    }

#### Пример ответа

    {
        "31.01.2021": 10066.67,
        "28.02.2021": 10133.78,
        "31.03.2021": 10201.34,
        "30.04.2021": 10269.35,
        "31.05.2021": 10337.81,
        "30.06.2021": 10406.73,
        "31.07.2021": 10476.11,
        "31.08.2021": 10545.95,
        "30.09.2021": 10616.26,
        "31.10.2021": 10687.04
    }

#### Пример ответа ошибки

    {
    "detail": {
        "error": "процентная ставка (rate) должна быть не меньше 1 и не превышать 8"
        }
    }

#### Команда для сборки контейнера

    docker build -t romirascoder/sber-api:dev .

#### Команда для запуска контейнера

    docker run -p 0.0.0.0:80:8000/tcp --name sber-api romirascoder/sber-api:dev

#### Команда для запуска тестов

    pytest
